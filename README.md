# Code-Conducted-and-Case-Study-Problems
GROL-Jaya: Golden Ratio-Based Oppositional Jaya Algorithm Large-Scale Time–Cost Trade-Off Optimization in Construction Projects
Authors Mohammad Azim Eirgash¹ PhD, Civil Engineering Department, Karadeniz Technical University, 61080 Trabzon, Türkiye Email: azim.eirgash@gmail.com
ORCID: https://orcid.org/0000-0001-5399-115X Victor Yepes² Prof., Institute of Concrete Science and Technology (ICITECH), Universitat Politècnica de València, 46022 València, Spain Email: vyepesp@cst.upv.es ORCID: https://orcid.org/0000-0001-5488-6001 Overview GROL-Jaya is a hybrid optimization framework combining the Jaya algorithm with Golden Ratio-Based Opposition Learning (GROL) to solve Time–Cost Trade-Off Problems (TCTPs) in construction project scheduling. It improves convergence, solution diversity, and scalability across small, medium, and large projects.
Case Studies The algorithm is validated using three construction scheduling case studies: Case 1: 7 activities (small-scale project) Case 2: 208 activities (medium-scale project) Case 3: 291 activities (large-scale project) These cases demonstrate the algorithm’s performance, Pareto-optimal solution quality, and scalability.
Performance & Decision-Making Metrics: Hypervolume (HV) and Spread (Sp) Decision-making: CRITIC–TOPSIS and ENTROPY–TOPSIS Example: PF7 is the preferred solution for the 208-activity case based on closeness coefficient (Ci).
